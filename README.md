# Paid-Media-Marketing-Science-Project
Conversion Response Modeling + Marginal ROAS Optimization 
Dataset: ppc_campaign_performance_data.csv 
Goal: Diagnose performance, fit a conversion response model with diminishing returns, derive marginal ROAS, and simulate budget reallocation
## What this project demonstrates
KPI validation & marketing metric literacy
Decomposition of CPA into CPC vs CVR
Conversion response modeling with diminishing returns (elasticity)
Translating a model into marginal ROAS
Budget reallocation simulation under diminishing returns
